## 项目简介
这个项目用了十分简单易懂的几行`css`、`js`实现了一个可交互的赛博手绘信封。  

项目已经考虑到了移动端的情况，所以做了自适应、响应式的布局方案。  

信封的徽章使用了@zhangyu1818 的开源项目 [appicon-forge](https://github.com/zhangyu1818/appicon-forge) 制作，闭合的时候效果如下
